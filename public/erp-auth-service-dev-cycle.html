Quality Gates
Unit Tests
Use cases aislados. Mocks de repos. Lógica de negocio 100%.
Integration Tests
PostgreSQL real (Testcontainers). Ciclo completo login→refresh→logout.
Security Tests
401 / 403 / RBAC deny. JWT tamperado. Token expirado.
Concurrency
Seat enforcement bajo concurrencia. SELECT FOR UPDATE. Race conditions.
Contract Tests · Error Model
Validar shape exacto: { code, message, requestId, traceId, timestamp, status } en todos los errores.
5
Security & Compliance
JWT HS256
accessToken 15 min · claims: sub, sid, roles, modules, permissions, iat, exp
Refresh rotation
prev_rt_hash ← actual → nuevo. Una única ventana de uso válida.
Reuse detection
Match con prev_rt_hash → revocar sesión + audit + 401.
Revocación
revoked_at por sid o por sub (logout/all).
Auditoría
actor_user_id · target · request_id · trace_id · metadata · created_at
Hardening
CORS strict · rate-limit · BCrypt cost ≥ 12 · no secrets en logs
